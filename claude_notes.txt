  How to run on Colab

  1. Upload this notebook + your data/ folder to a Colab session (or to a Drive folder
  named asl_project).
  2. Runtime → Change runtime type → GPU (L4 or A100 if available).
  3. Run §1 cells. Verify DEVICE=cuda.
  4. Run §2 (small). The Kaggle download cell is gated behind # external_df =
  load_external() — uncomment to download.
  5. Section §4.2 has run_all_configs() commented-out; uncomment to launch the full
  training sweep. Total wall-clock estimate on a single L4 GPU: ~6h for all six configs
  (MS3-ref ResNet-18 ~5min, ResNet-18@224 ~25min, ResNet-50 strong ~80min, ResNet-50
  mild ~80min, ViT-B/16 ~90min, DINOv2 ~60min). On A100 cut roughly in half.
  6. After training, all §5 and §6 cells consume the saved runs/<name>/ artifacts and
  run quickly (probing + TCAV are CPU-OK; counterfactual + saliency are tiny per-image
  GPU calls).

  The notebook is resumable: every run skips if ckpt_best.pt exists; if Colab
  disconnects mid-training, just rerun §4.2 — only the unfinished config gets retried.
